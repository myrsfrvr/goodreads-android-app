# Book Tracking App

This is a basic Android app designed to track books using the Goodreads API. The app uses Kotlin, Jetpack Compose, and follows the MVVM (Model-View-ViewModel) architecture.

## Features:
- **Tab 1: Book Recommendations** – Displays book recommendations using the Goodreads API.
- **Tab 2: Book Search** – Allows users to search for books.
- **Tab 3: User's Books** – Displays the books that the user has read or wants to read.

---

## Tech Stack
- **Kotlin**: Programming language used for Android development.
- **Jetpack Compose**: UI toolkit for building native Android UIs.
- **MVVM**: Architecture pattern used for separating business logic from UI code.
- **Goodreads API**: External API used to fetch book data (for recommendations, search, etc.).

---

## Structure
- **Model**: Represents the data, such as books and user preferences.
- **View**: UI components created with Jetpack Compose.
- **ViewModel**: Manages UI-related data and communicates with the model layer.

---

## Setup

1. Clone the repository.
2. Open the project in Android Studio.
3. Make sure you have the required dependencies and API keys for accessing the Goodreads API.
4. Build and run the app.

## Czech (Čeština)

# Aplikace pro sledování knih

Toto je základní aplikace pro Android, která slouží k sledování knih pomocí Goodreads API. Aplikace používá Kotlin, Jetpack Compose a dodržuje architekturu MVVM (Model-View-ViewModel).

## Funkce:
- **Tab 1: Doporučení knih** – Zobrazuje doporučení knih prostřednictvím Goodreads API.
- **Tab 2: Vyhledávání knih** – Umožňuje uživatelům vyhledávat knihy.
- **Tab 3: Uživatelské knihy** – Zobrazuje knihy, které uživatel přečetl nebo chce přečíst.

---

## Technologický stack
- **Kotlin**: Programovací jazyk používaný pro vývoj pro Android.
- **Jetpack Compose**: Nástroj pro tvorbu uživatelského rozhraní v nativní aplikaci pro Android.
- **MVVM**: Architektonický vzor pro oddělení byznys logiky od kódu uživatelského rozhraní.
- **Goodreads API**: Externí API pro získávání dat o knihách (doporučení, vyhledávání, atd.).

---

## Struktura
- **Model**: Reprezentuje data, jako jsou knihy a uživatelské preference.
- **View**: Komponenty uživatelského rozhraní vytvořené pomocí Jetpack Compose.
- **ViewModel**: Spravuje data související s uživatelským rozhraním a komunikuje s vrstvou modelu.
